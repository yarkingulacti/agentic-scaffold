# Contributing

Thanks for considering a contribution.

## Issues

- Open a [GitHub Issue](https://github.com/yarkingulacti/agentic-scaffold/issues) for bugs, feature requests, or questions.
- Use the issue templates when available.

## Pull requests

1. Open an issue first to discuss the change you want to make.
2. Branch from `master` using a conventional branch name:
   - `feat/<description>` for new features
   - `fix/<description>` for bug fixes
   - `docs/<description>` for documentation
   - `refactor/<description>` for refactoring
3. Commit using [Conventional Commits](https://www.conventionalcommits.org/):

   ```
   <Short summary of the change>

   <type>(<scope>): <imperative description>
   ```

4. Open a PR into `master` with a clear description.

## Code of conduct

Be respectful and constructive. Harassment and offensive behaviour will not be
tolerated.
